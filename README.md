# Abdulaziz Wakah Portfolio

Modern personal portfolio for Abdulaziz Wakah, a full-stack developer with a backend-first focus.

The site presents selected work, internship experience, technical strengths, contact links, and project details in a polished dark responsive layout.

## Highlights

- Full-stack developer positioning with strong backend emphasis
- Selected work section with live project and GitHub links
- Custom backend architecture preview for the e-commerce backend project
- Responsive dark UI for desktop and mobile
- Personal profile card and project visuals
- Content centralized in `src/content/portfolio.js`

## Tech Stack

- React
- Vite
- CSS
- Lucide React icons

## Projects Featured

- **AFAQ Talents Daycare** - React/Vite daycare landing page with playful visual design.
- **E-Commerce Backend** - NestJS, TypeScript, PostgreSQL, TypeORM, S3, and custom authentication.
- **Personal Portfolio** - This portfolio, built as a responsive full-stack developer portfolio.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
